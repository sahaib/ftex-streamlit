# FAQ

## General

**What is FTEX?**
Open-source ticket analytics platform with AI-powered insights.

**Is it free?**
Yes, 100% free under MIT license.

**What helpdesks are supported?**
Freshdesk (API), Zendesk (export), any JSON/CSV system.

## Installation

**System requirements?**
Python 3.9+, 4GB RAM, 1GB storage.

**Can I run without Docker?**
Yes: `pip install -r requirements.txt && streamlit run app/main.py`

## Data

**What formats are supported?**
JSON (Freshdesk export), CSV.

**File size limits?**
500MB default, configurable.

## AI

**Which AI providers work?**
Ollama (local), OpenAI, Anthropic.

**Is AI required?**
No, FTEX works fully without AI.
