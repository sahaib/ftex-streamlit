# Dashboard Guide

The main analytics dashboard provides a comprehensive overview.

## KPI Cards

- **Total Tickets** - All tickets in current filter
- **SLA Compliance** - % meeting first response target
- **Open Tickets** - Currently unresolved
- **Stale Tickets** - Open > threshold (default 15 days)

## Charts

### Ticket Volume Trend
Daily/weekly ticket creation. Identify spikes and patterns.

### Status Distribution
Pie chart: Open, Pending, Resolved, Closed.

### Category Breakdown
Top issue categories by volume.

### Priority Mix
Urgent, High, Medium, Low distribution.

## Filters

Use sidebar filters:

- **Date Range** - Start/end dates
- **Company** - Specific customer
- **Status** - Ticket status
- **Priority** - Urgency level

All charts update automatically.
